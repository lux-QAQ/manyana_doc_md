# Windows部署教程  
由于部署教程视频在[B站](https://www.bilibili.com/video/BV1MS421R7PV?p=2)上已经很详细了，这里不做过多介绍。
::: danger 需要注意的是:boom:
1. Fluent版本的UI启动器由于开发问题，没有图形化修改配置文件的界面，只能手动去配置文件里面修改   
   （这容易导致格式错误而无法启动，如果你打算修改配置文件，请备份，或者弄明白 yaml 的格式）
2. Fluent版本的UI启动器所支持的 Onebot 协议暂时还能用。
3. 基于以上原因，建议您使用 PYQT版本启动器。
:::
